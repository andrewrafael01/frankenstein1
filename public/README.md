# Public Assets

## Courtroom Background Image

Place a courtroom image file named `courtroom.jpg` in this directory for the 3D courtroom visual.

The image should be:
- A courtroom scene from a judge's perspective
- Preferably dark/moody to match the gothic aesthetic
- Will be automatically styled with grayscale and darkening filters

If no image is provided, the app will use a gradient fallback background.
